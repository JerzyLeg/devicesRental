Detailed Description

Klasa obslugujaca rezerwacje.

Klasa posiada wektory najwazniejszych skladowych wypozyczalni - rezerwacji, uzytkonikow i urzadzen oraz najwazniejsze metody umozliwiajace jej dzialanie.
